<h3> 利用Python反射功能，根据配置文件动态加载类&初始化类</h3>
适用于Python2.7.5

 - test_class.py   
定义基本的测试类/基类   

 - config.yaml, config.py    
类加载yaml配置文件以及读取yaml文件的工具    

 - construct.py   
根据类的路径import类，并根据配置组装、初始化所有的基类
